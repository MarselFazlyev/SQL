БД - совокупность данных, которые хранятся по определенным правилам и используются для удовлетворения информационных потребностей пользователей.
Реляционная БД - это БД, где вся информация представляется в виде таблиц.
User - лицо, которое может подключиться к БД(log on process).
DB схема - это все объекты в БД,еоторые принадлежат одному User-у.

SQL команды делятся на 4 группы :

1)  DML(data manipulation language)  - команды, который работают с "внутренностями" таблиц. 

    SELECT - самая главная команда -  (похож на геттер в Java) 
    INSERT - аналог сеттера в Java
    UPDATE - обновление данных
    DELETE - удаление строк
    MERGE  - объединение трех функциональностей команд INSERT,UPDATE, DELETE

2) DDL(data definition language) - команды, работающие со структурой таблицы
   
   CREATE - создание таблицы
   ALTER  - изменяет структуру таблицы (например, можно добавить новый столбец)
   DROP   - удаляет объекты
   RENAME - изменение имени (например, изменить имя столбца)  
   TRUNCATE - удаление всех данных из таблицы

3) TCL (transaktion control language) - контроль транзакций

   COMMIT    - утвердить изменения 
   ROLLBACK  - откатить 
   SAVEPOINT - точка сохранения

4) DCL(data control language) - управление правами на использование, изменение табоиц (обычно нужны для DBA- data base administrator) 

   GRANT - с помощью этой команды могут дать право на просмотр изменение данных в таблице.
   REVOKE - с помощью этой команды могут отобрать права ...

ТИП ДАННЫХ - это множество допустимых значений этих данных, а также совокупность операций над ними.
   INTEGER  - 
   NUMBER(p,s) - p - общее максимальное количество цифр, s - мколько из бщего количества цифр отводится на дробную часть.
                 Максимальное количество цифр на целую часть числа = p-s
